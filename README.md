# 形象探索平台 - 前端网页

## 文件说明

- `index.html` - 主页面HTML文件
- `styles.css` - 样式表文件
- `script.js` - JavaScript交互文件
- `套餐命名体系配置.json` - 套餐数据配置文件
- `平台套餐命名体系规范.md` - 命名规范文档

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可查看网页
2. 或者使用本地服务器运行（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```
   然后在浏览器访问 `http://localhost:8000`

## 功能特点

- ✅ 响应式设计，支持移动端和桌面端
- ✅ 现代化的UI设计
- ✅ 平滑滚动导航
- ✅ 表单提交处理
- ✅ 页面加载动画效果
- ✅ 完全遵循命名体系规范

## 页面结构

1. **导航栏** - 固定顶部导航
2. **首页横幅** - 平台主题展示
3. **核心套餐服务** - 三大主线套餐系列
   - 初次体验系列
   - 形象打造系列
   - 身心护理系列
4. **进阶组合套餐** - 组合套餐展示
5. **服务原则** - 平台服务理念
6. **关于我们** - 平台介绍
7. **联系我们** - 联系表单
8. **页脚** - 版权信息

## 自定义配置

如需修改套餐内容，可以：
1. 直接编辑 `index.html` 中的套餐卡片
2. 或使用 `套餐命名体系配置.json` 中的数据动态生成页面

## 注意事项

- 所有命名严格遵循规范文档中的要求
- 避免使用禁止的关键词
- 保持专业、安全、尊重的服务定位
